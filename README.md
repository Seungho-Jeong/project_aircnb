# aircnb
airbnb 서비스를 클론한 프로젝트입니다.<br>본 레포지토리는 서버사이드 결과물 저장소입니다.

### Features
- 숙소 예약 및 예약 이메일 발송
- 숙소 찜하기
- 고객 가입정보 유효성 검사(Validation)
- Google, Kakao 소셜 로그인
- 비밀번호 암호화(Bcrypt)
- 숙소 종류, 숙박 유형, 어메니티 옵션 필터링에 따른 숙소목록 출력
- 숙소 목록 페이지네이션
- 숙소 상세 정보 출력
- Google map API를 활용한 좌표 및 마커설정
- 고객 후기(댓글), 숙소 평가점수 및 평균점수 현황 출력
